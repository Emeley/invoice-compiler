# invoice-compiler

Generate invoices from YAML files using XeLaTeX.

## Usage
```bash
./index.coffee invoice.yml
# Outputs invoice.pdf
```

See `samples` directory.


## Dependencies
* node & npm
* xelatex
* Open Sans font
