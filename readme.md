# invoice-compiler

Generate invoices from YAML files using [wkhtmltopdf](http://wkhtmltopdf.org/).

## Usage
```bash
./index.coffee invoice.yml
# Outputs invoice.pdf
```

See `samples` directory.


## Dependencies
* node & npm
* Open Sans font
